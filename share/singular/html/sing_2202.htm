D.15.14.37 isHomogeneous

Procedure from library multigrading.lib (see multigrading_lib).

Usage:
isHomogeneous(a[, f]); a polynomial/vector/ideal/module

Return:
boolean, TRUE if a is (multi)homogeneous, and FALSE otherwise
