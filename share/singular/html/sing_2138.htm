D.15.9.3 gitFan

Procedure from library gitfan.lib (see gitfan_lib).

Purpose:
Returns the GIT-fan of the H-action defined by Q on V(a). An optional third parameter of type 'int' is interpreted as the number of CPU-cores you would like to use.
Note that 'system("--cpus");' returns the number of cpu available in your system.

Return:
a fan.

Example:

intmat Q[3][4] =
1,0,1,0,
0,1,0,1,
0,0,1,1;
ring R = 0,T(1..4),dp;
ideal a = 0;
gitFan(a, Q);
