7.2.7 ring (plural)

Rings are used to describe properties of polynomials, ideals etc. Almost all computations in PLURAL require a basering. For a detailed description of the concept of rings see Rings and orderings.


Note: PLURAL usually works with global orderings (see PLURAL) but one can use certain local once when graded commutative rings are being used.
