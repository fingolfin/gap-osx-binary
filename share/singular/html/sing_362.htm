5.1.103 nvars

Syntax:
nvars ( ring_name )
Type:
int
Purpose:
returns the number of variables of a ring.
Example:
 
  ring r=(23,t,v),(x,a(1..7)),ls;
  // the variables are x,a(1),...,a(7)
  nvars(r);
==> 8
See npars; ring; var; varstr.
