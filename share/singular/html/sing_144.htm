4.14 package

The data type package is used to group identifiers into collections. It is mainly used as an internal means to avoid collisions of names of identifiers in libraries with variable names defined by the user. The most important package is the toplevel package, called Top. It contains all user defined identifiers as well as all user accessible library procedures. Identifiers which are local to a library are contained in a package whose name is obtained from the name of the library, where the first letter is converted to uppercase, the remaining ones to lowercase. Another reserved package name is Current which denotes the current package name in use. See also Libraries.
