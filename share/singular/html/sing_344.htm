5.1.85 min

Procedure from library standard.lib (see standard_lib).

Syntax:
max (i, j)

Type:
same as type of i resp. j

Purpose:
returns the maximum for any 2 arguments of a type
for which '>' is defined

Example:
 
  max(2,3);
==> 3
max(4,3);
==> 4
See also: min.
