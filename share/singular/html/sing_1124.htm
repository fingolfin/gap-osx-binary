D.4.3.2 assPrimes

Procedure from library assprimeszerodim.lib (see assprimeszerodim_lib).

Assume:
I is zero-dimensional over Q[variables]

Return:

Note:
A final test is applied to the result if exactness != 0 (default), otherwise no final test is done.

2fd+e2+2ca+c+b2,

2fa+f+2eb+2dc;
