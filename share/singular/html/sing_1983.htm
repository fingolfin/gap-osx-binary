D.13.3.20 texMatrix

Procedure from library tropical.lib (see tropical_lib).

Usage:
texMatrix(M); M matrix

Return:
string, the tex command representing M

Example:
 
LIB "tropical.lib";
ring r=(0,t),x,dp;
matrix M[2][2]=3/2,1/t*x2-t2x+1/t,5,-2x;
texMatrix(M);
