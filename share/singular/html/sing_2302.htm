D.16.2.18 isNormal

Syntax:
Polymake::isNormal( polytope p )
Type:
int
Purpose:
1 if the projective toric variety defined by p is projectively normal, 0 otherwise.
Example:
 
  LIB"polymake.lib";
==> Welcome to polymake

  polytope p = polytopeViaPoints(M);
  Polymake::isNormal(p);
