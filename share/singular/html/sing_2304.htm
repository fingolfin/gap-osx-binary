D.16.2.20 isSmooth

Syntax:
Polymake::isSmooth( cone c )
Polymake::isSmooth( fan f )
Polymake::isSmooth( polytope p )
Type:
int
Purpose:
1 if c, f or p is smooth, 0 otherwise.
Example:
 
  LIB"polymake.lib";
