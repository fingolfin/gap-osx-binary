Library:
finitediff.lib
Purpose:
procedures to compute finite difference schemes for linear differential equations
Author:
Christian Dingler

Overview:
Using qepcad/qepcadsystem from this library requires the program qepcad to be installed. You can download qepcad from http://www.usna.edu/Users/cs/qepcad/INSTALL/IQ.html
