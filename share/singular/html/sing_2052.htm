D.15.2.24 arrTypeB

Procedure from library arr.lib (see arr_lib).

Usage:
arrTypeB(v); int v

Return:
arrangement, which uses first v variables of ring for reflection arrangement of type B

Example:
 
LIB "arr.lib";
ring R = 0,x(1..10),dp;
arrTypeB(5);
==> _[1]=x(1)-x(2)
==> _[2]=x(1)+x(2)
==> _[3]=x(1)-x(3)
==> _[4]=x(1)+x(3)
==> _[5]=x(1)-x(4)
==> _[6]=x(1)+x(4)
==> _[7]=x(1)-x(5)
==> _[8]=x(1)+x(5)
==> _[9]=x(1)
==> _[10]=x(2)-x(3)
==> _[11]=x(2)+x(3)
==> _[12]=x(2)-x(4)
==> _[13]=x(2)+x(4)
==> _[14]=x(2)-x(5)
==> _[15]=x(2)+x(5)
==> _[16]=x(2)
==> _[17]=x(3)-x(4)
==> _[18]=x(3)+x(4)
==> _[19]=x(3)-x(5)
==> _[20]=x(3)+x(5)
==> _[21]=x(3)
==> _[22]=x(4)-x(5)
==> _[23]=x(4)+x(5)
==> _[24]=x(4)
==> _[25]=x(5)
==>
