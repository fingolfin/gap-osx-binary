4.22.3.21 isOrigin

Syntax:
isOrigin( cone c )
Type:
int
Purpose:
1 iff the cone is just the origin of the ambient space; 0 otherwise
Example:
 
LIB"gfanlib.so";
cone c1; // full cone in zero-dimensional ambient space
isOrigin(c1);
==> 1
intmat M2[2][2]=
1,0,
0,1;
cone c2=coneViaPoints(M2);
isOrigin(c2);
==> 0
intmat M3[4][2]=
1,0,
0,1,
-1,0,
0,-1;
cone c3=coneViaPoints(M3);
isOrigin(c3);
==> 0
