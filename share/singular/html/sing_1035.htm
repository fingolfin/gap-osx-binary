D.2.8.18 hasCommutativeVars

Procedure from library ring.lib (see ring_lib).

Usage:
hasCommutativeVars ( rng );

Return:
1 if rng is a commutative polynomial ring, 0 otherwise.

Example:
 
LIB "ring.lib";
ring r=0,(x,y,z),dp;
hasCommutativeVars(r);
==> 1
