D.5.2.14 resfunction

Procedure from library resbinomial.lib (see resbinomial_lib).

Usage:
resfunction(invariant,auxinv,nchart,n);
invariant, auxinv lists, nchart, n integers

Compute:
Patch the resolution function

Return:
The complete resolution function

Example:
 
LIB "resbinomial.lib";
ring r = 0,(x(1..2)),dp;
ideal J=x(1)^2-x(2)^3;
list L=Eresol(J);
L[3]; // incomplete resolution function
