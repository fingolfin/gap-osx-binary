D.15.9 gitfan_lib

Library:
gitfan.lib
Purpose:
Compute GIT-fans.

Authors:
Janko Boehm boehm@mathematik.uni-kl.de
Simon Keicher keicher@mail.mathematik.uni-tuebingen.de
Yue Ren ren@mathematik.uni-kl.de

Overview:
This library computes GIT-fans, torus orbits and GKZ-fans. It uses the package 'gfanlib' by Anders N. Jensen
and some algorithms have been outsourced to C++ to improve the performance. Check https://github.com/skeicher/gitfan_singular for updates.

Procedures:

D.15.9.1 afaces  Returns a list of intvecs that correspond to all a-faces
D.15.9.2 gitCone  Returns the GIT-cone around the given weight vector w
D.15.9.3 gitFan  Returns the GIT-fan of the H-action defined by Q on V(a)
D.15.9.4 gkzFan  Returns the GKZ-fan of the matrix Q
D.15.9.5 isAface  Checks whether intvec corresponds to an ideal-face
D.15.9.6 orbitCones  Returns the list of all projected a-faces
