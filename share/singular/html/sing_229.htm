4.23.1.7 insertCone

Syntax:
insertCone( fan f, cone c )
insertCone( fan f, cone c, int check )
Type:
none
Purpose:
inserts the cone into the fan; checks for compatibility beforehand unless check is passed and equal 0
Example:
 
LIB"gfanlib.so";
fan f=emptyFan(3);
f;
