D.15.17 parallel_lib

Library:
parallel.lib
Purpose:
An abstraction layer for parallel skeletons

Author:
Andreas Steenpass, e-mail: steenpass@mathematik.uni-kl.de

Overview:
This library provides implementations of several parallel 'skeletons' (i.e. ways in which parallel tasks rely upon and interact with each other). It is based on the library tasks.lib and aims at both ordinary Singular users as well as authors of Singular libraries.

Procedures:

D.15.17.1 parallelWaitN  execute several jobs in parallel and wait for N of them to finish
D.15.17.2 parallelWaitFirst  execute several jobs in parallel and wait for the first to finish
D.15.17.3 parallelWaitAll  execute several jobs in parallel and wait for all of them to finish
D.15.17.4 parallelTestAND  run several tests in parallel and determine if they all succeed
D.15.17.5 parallelTestOR  run several tests in parallel and determine if any of them succeeds
