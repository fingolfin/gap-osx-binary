  [1X6.1 [33X[0;0YStraight Line Decisions[133X[101X

  list of standard generators (cf.Â Section[14X3.3[114X) for this group.[133X
  
  [33X[0;0YA  straight  line  decision in [5XGAP[105X is represented by an object in the filter
  [2XIsStraightLineDecision[102X ([14X6.1-1[114X) that stores a list of [21Xlines[121X each of which has
  one of the following three forms.[133X

  [31X2[131X   [33X[0;6Ya  pair  [22X[  l,  i  ][122X  where [22Xl[122X is a list of form 1. and [22Xi[122X is a positive

  [33X[0;0YFor  the  meaning  of  the  list  of lines, see [2XResultOfStraightLineDecision[102X

  ([14X6.1-2[114X),     an     operation     for    straight    line    decisions    is

  [1X6.1-6 ResultOfStraightLineDecision[101X
  
  [29X[2XResultOfStraightLineDecision[102X( [3Xprog[103X, [3Xgens[103X[, [3Xorderfunc[103X] ) [32X operation
  [6XReturns:[106X  [33X[0;10Y[9Xtrue[109X if all checks succeed, otherwise [9Xfalse[109X.[133X
  
  [33X[0;0Y[2XResultOfStraightLineDecision[102X    evaluates   the   straight   line   decision
  (seeÂ [2XIsStraightLineDecision[102X  ([14X6.1-1[114X)) [3Xprog[103X at the group elements in the list
  [3Xgens[103X.[133X
  
  [33X[0;0YThe  function for computing the order of a group element can be given as the
  optional  argument [3Xorderfunc[103X. For example, this may be a function that gives
  up  at  a  certain  limit if one has to be aware of extremely huge orders in
  failure cases.[133X
  
  [33X[0;0YThe  [13Xresult[113X  of  a straight line decision with lines [22Xp_1, p_2, ..., p_k[122X when
  applied to [3Xgens[103X is defined as follows.[133X
  
  [8X(a)[108X
        [33X[0;6YFirst  a  list  [22Xr[122X of intermediate values is initialized with a shallow
        copy of [3Xgens[103X.[133X
  
  [8X(b)[108X
        [33X[0;6YFor  [22Xi  â‰¤ k[122X, before the [22Xi[122X-th step, let [22Xr[122X be of length [22Xn[122X. If [22Xp_i[122X is the
        external  representation  of  an  associative  word  in  the  first  [22Xn[122X
        generators  then the image of this word under the homomorphism that is
        given  by  mapping [22Xr[122X to these first [22Xn[122X generators is added to [22Xr[122X. If [22Xp_i[122X
        is  a  pair [22X[ l, j ][122X, for a list [22Xl[122X, then the same element is computed,
        but  instead  of being added to [22Xr[122X, it replaces the [22Xj[122X-th entry of [22Xr[122X. If
        [22Xp_i[122X  is a triple [22X[[122X[10X"Order"[110X[22X, i, n ][122X then it is checked whether the order
        of [22Xr[i][122X is [22Xn[122X; if not then [9Xfalse[109X is returned immediately.[133X
  
  [8X(c)[108X
        [33X[0;6YIf  all [22Xk[122X lines have been processed and no order check has failed then
        [9Xtrue[109X is returned.[133X
  
  [33X[0;0YHere are some examples.[133X
  
  [4X[32X  Example  [32X[104X

    [4X[28Xtrue[128X[104X

  [33X[0;0YThe  above  straight  line  decision  [10Xdec[110X returns [9Xtrue[109X â€“for [13Xany[113X input of the

  [1X6.1-7 [33X[0;0YSemi-Presentations and Presentations[133X[101X
  
  [33X[0;0YWe  can  associate  a  [13Xfinitely  presented group[113X [22XF / R[122X to each straight line
  decision  [3Xdec[103X,  say, as follows. The free generators of the free group [22XF[122X are
  in  bijection  with  the inputs, and the defining relators generating [22XR[122X as a
  normal  subgroup  of [22XF[122X are given by those words [22Xw^k[122X for which [3Xdec[103X contains a
  check whether the order of [22Xw[122X equals [22Xk[122X.[133X
  
  [33X[0;0YSo  if  [3Xdec[103X  returns  [9Xtrue[109X  for  the  input list [22X[ g_1, g_2, ..., g_n ][122X then
  mapping the free generators of [22XF[122X to the inputs defines an epimorphism [22XÎ¦[122X from
  [22XF[122X  to  the  group  [22XG[122X, say, that is generated by these inputs, such that [22XR[122X is
  contained in the kernel of [22XÎ¦[122X.[133X
  
  [33X[0;0Y(Note  that  [21Xsatisfying  [3Xdec[103X[121X  is  a  stronger  property  than  [21Xsatisfying  a
  presentation[121X.  For  example, [22XâŸ¨ x âˆ£ x^2 = x^3 = 1 âŸ©[122X is a presentation for the
  trivial  group, but the straight line decision that checks whether the order
  of [22Xx[122X is both [22X2[122X and [22X3[122X clearly always returns [9Xfalse[109X.)[133X
  
  [33X[0;0YThe  [5XATLAS[105X  of  Group  Representations  contains  the following two kinds of
  straight line decisions.[133X
  
  [30X    [33X[0;6YA  [13Xpresentation[113X  is a straight line decision [3Xdec[103X that is defined for a
        set  of  standard generators of a group [22XG[122X and that returns [9Xtrue[109X if and
        only  if  the  list  of  inputs is in fact a sequence of such standard
        generators  for [22XG[122X. In other words, the relators derived from the order
        checks  in  the  way  described above are defining relators for [22XG[122X, and
        moreover these relators are words in terms of standard generators. (In
        particular  the  kernel  of  the  map  [22XÎ¦[122X equals [22XR[122X whenever [3Xdec[103X returns
        [9Xtrue[109X.)[133X
  
  [30X    [33X[0;6YA  [13Xsemi-presentation[113X  is  a straight line decision [3Xdec[103X that is defined
        for  a  set  of standard generators of a group [22XG[122X and that returns [9Xtrue[109X
        for a list of inputs [13Xthat is known to generate a group isomorphic with
        [22XG[122X[113X  if  and  only  if  these inputs form in fact a sequence of standard
        generators  for [22XG[122X. In other words, the relators derived from the order
        checks  in  the  way  described  above  are  [13Xnot  necessarily defining
        relators[113X for [22XG[122X, but if we assume that the [22Xg_i[122X generate [22XG[122X then they are
        standard  generators. (In particular, [22XF / R[122X may be a larger group than
        [22XG[122X  but  in  this  case  [22XÎ¦[122X  maps  the  free generators of [22XF[122X to standard
        generators of [22XG[122X.)[133X
  
        [33X[0;6YMore about semi-presentations can be found in [NW05].[133X
  
  [33X[0;0YAvailable    presentations    and    semi-presentations    are   listed   by
  [2XDisplayAtlasInfo[102X  ([14X3.5-1[114X),  they  can  be accessed via [2XAtlasProgram[102X ([14X3.5-3[114X).
  (Clearly   each   presentation   is   also   a   semi-presentation.   So   a
  semi-presentation  for  some  standard  generators of a group is regarded as

  [1X6.1-9 StraightLineProgramFromStraightLineDecision[101X

  [1X6.2 [33X[0;0YBlack Box Programs[133X[101X
  
  [33X[0;0Y[13XBlack  box  programs[113X  formalize the idea that one takes some group elements,
  forms  arithmetic  expressions  in  terms of them, tests properties of these
  expressions,  executes  conditional  statements  (including jumps inside the
  program)  depending  on  the  results of these tests, and eventually returns
  some result.[133X
  
  [33X[0;0YA specification of the language can be found in [Nic06], see also[133X
  
  [33X[0;0Y[7Xhttp://brauer.maths.qmul.ac.uk/Atlas/info/blackbox.html[107X.[133X
  
  [33X[0;0YThe  [13Xinputs[113X  of  a black box program may be explicit group elements, and the
  program  may  also  ask  for random elements from a given group. The [13Xprogram
  steps[113X  form  products,  inverses,  conjugates,  commutators,  etc.  of known
  elements,  [13Xtests[113X  concern essentially the orders of elements, and the [13Xresult[113X
  is a list of group elements or [9Xtrue[109X or [9Xfalse[109X or [9Xfail[109X.[133X
  
  [33X[0;0YExamples that can be modeled by black box programs are[133X
  
  [8X[13Xstraight line programs[113X,[108X
        [33X[0;6Ywhich  require  a  fixed  number of input elements and form arithmetic
        expressions  of  elements  but  do  not  use  random  elements, tests,
        conditional statements and jumps; the return value is always a list of
        elements; these programs are described in Section [14X'Reference: Straight
        Line Programs'[114X.[133X
  
  [8X[13Xstraight line decisions[113X,[108X
        [33X[0;6Ywhich  differ  from straight line programs only in the sense that also
        order  tests  are admissible, and that the return value is [9Xtrue[109X if all
        these  tests  are  satisfied, and [9Xfalse[109X as soon as the first such test
        fails; they are described in Section [14X6.1[114X.[133X
  
  [8X[13Xscripts for finding standard generators[113X,[108X
        [33X[0;6Ywhich take a group and a function to generate a random element in this
        group  but  no  explicit input elements, admit all control structures,
        and  return  either  a  list  of  standard  generators  or  [9Xfail[109X;  see
        [2XResultOfBBoxProgram[102X ([14X6.2-4[114X) for examples.[133X
  
  [33X[0;0YIn  the  case of general black box programs, currently [5XGAP[105X provides only the
  possibility  to read an existing program via [2XScanBBoxProgram[102X ([14X6.2-2[114X), and to
  run  the  program  using  [2XRunBBoxProgram[102X ([14X6.2-3[114X). It is not our aim to write
  such programs in [5XGAP[105X.[133X
  
  [33X[0;0YThe  special  case of the [21Xfind[121X scripts mentioned above is also admissible as
  an  argument of [2XResultOfBBoxProgram[102X ([14X6.2-4[114X), which returns either the set of
  generators or [9Xfail[109X.[133X
  
  [33X[0;0YContrary  to  the  general  situation, more support is provided for straight
  line  programs  and  straight line decisions in [5XGAP[105X, see Section [14X'Reference:
  Straight  Line  Programs'[114X  for  functions  that  manipulate  them  (compose,
  restrict etc.).[133X
  
  [33X[0;0YThe   functions  [2XAsStraightLineProgram[102X  ([14X6.2-6[114X)  and  [2XAsStraightLineDecision[102X
  ([14X6.1-8[114X)  can  be used to transform a general black box program object into a
  straight line program or a straight line decision if this is possible.[133X
  
  [33X[0;0YConversely,  one  can  create an equivalent general black box program from a
  straight  line  program  or from a straight line decision with [2XAsBBoxProgram[102X
  ([14X6.2-5[114X).[133X
  
  [33X[0;0Y(Computing a straight line program related to a given straight line decision
  is  supported  in  the  sense of [2XStraightLineProgramFromStraightLineDecision[102X
  ([14X6.1-9[114X).)[133X
  
  [33X[0;0YNote that none of these three kinds of objects is a special case of another:
  Running  a  black  box  program with [2XRunBBoxProgram[102X ([14X6.2-3[114X) yields a record,
  running a straight line program with [2XResultOfStraightLineProgram[102X ([14XReference:
  ResultOfStraightLineProgram[114X)  yields  a  list  of  elements,  and  running a
  straight line decision with [2XResultOfStraightLineDecision[102X ([14X6.1-6[114X) yields [9Xtrue[109X
  or [9Xfalse[109X.[133X
  
  [1X6.2-1 IsBBoxProgram[101X
  
  [29X[2XIsBBoxProgram[102X( [3Xobj[103X ) [32X Category

  [1X6.2-2 ScanBBoxProgram[101X
  
  [29X[2XScanBBoxProgram[102X( [3Xstring[103X ) [32X function

    [4X[25X>[125X [27X  if B notin 1 2 3 5 then fail\n\[127X[104X
    [4X[25X>[125X [27X  if B noteq 3 then jmp START2\n\[127X[104X

  [1X6.2-3 RunBBoxProgram[101X
  
  [29X[2XRunBBoxProgram[102X( [3Xprog[103X, [3XG[103X, [3Xinput[103X, [3Xoptions[103X ) [32X function
  [6XReturns:[106X  [33X[0;10Ya  record  describing the result and the statistics of running the
            black box program [3Xprog[103X, or [9Xfail[109X, or the string [10X"timeout"[110X.[133X
  
  [33X[0;0YFor a black box program [3Xprog[103X, a group [3XG[103X, a list [3Xinput[103X of group elements, and
  a record [3Xoptions[103X, [2XRunBBoxProgram[102X applies [3Xprog[103X to [3Xinput[103X, where [3XG[103X is used only
  to compute random elements.[133X
  
  [33X[0;0YThe  return value is [9Xfail[109X if a syntax error or an explicit [10Xfail[110X statement is
  reached  at  runtime,  and  the  string  [10X"timeout"[110X if a [10Xtimeout[110X statement is
  reached.  (The  latter  might  mean  that  the random choices were unlucky.)
  Otherwise a record with the following components is returned.[133X
  
  [8X[10Xgens[110X[108X
        [33X[0;6Ya list of group elements, bound if an [10Xoup[110X statement was reached,[133X
  
  [8X[10Xresult[110X[108X
        [33X[0;6Y[9Xtrue[109X  if  a  [10Xtrue[110X  statement  was  reached,  [9Xfalse[109X  if  either a [10Xfalse[110X
        statement or a failed order check was reached,[133X
  
  [33X[0;0YThe  other  components serve as statistical information about the numbers of
  the  various  operations ([10Xmultiply[110X, [10Xinvert[110X, [10Xpower[110X, [10Xorder[110X, [10Xrandom[110X, [10Xconjugate[110X,
  [10Xconjugateinplace[110X, [10Xcommutator[110X), and the runtime in milliseconds ([10Xtimetaken[110X).[133X
  
  [33X[0;0YThe following components of [3Xoptions[103X are supported.[133X
  
  [8X[10Xrandomfunction[110X[108X
        [33X[0;6Ythe  function  called  with  argument  [3XG[103X  in order to compute a random
        element of [3XG[103X (default [2XPseudoRandom[102X ([14XReference: PseudoRandom[114X))[133X
  
  [8X[10Xorderfunction[110X[108X
        [33X[0;6Ythe  function  for  computing  element  orders  (the  default is [2XOrder[102X
        ([14XReference: Order[114X)),[133X
  
  [8X[10Xquiet[110X[108X
        [33X[0;6Yif [9Xtrue[109X then ignore [10Xecho[110X statements (default [9Xfalse[109X),[133X
  
  [8X[10Xverbose[110X[108X
        [33X[0;6Yif  [9Xtrue[109X  then  print  information  about  the  line that is currently
        processed, and about order checks (default [9Xfalse[109X),[133X
  
  [8X[10Xallowbreaks[110X[108X
        [33X[0;6Yif  [9Xtrue[109X  then call [2XError[102X ([14XReference: Error[114X) when a [10Xbreak[110X statement is
        reached, otherwise ignore [10Xbreak[110X statements (default [9Xtrue[109X).[133X
  
  [33X[0;0YAs  an example, we run the black box programs constructed in the example for
  [2XScanBBoxProgram[102X ([14X6.2-2[114X).[133X
  
  [4X[32X  Example  [32X[104X
    [4X[25Xgap>[125X [27Xg:= AlternatingGroup( 5 );;[127X[104X
    [4X[25Xgap>[125X [27Xres:= RunBBoxProgram( find.program, g, [], rec() );;[127X[104X
    [4X[25Xgap>[125X [27XIsBound( res.gens );  IsBound( res.result );[127X[104X
    [4X[28Xtrue[128X[104X
    [4X[28Xfalse[128X[104X
    [4X[25Xgap>[125X [27XList( res.gens, Order );[127X[104X
    [4X[28X[ 2, 3 ][128X[104X
    [4X[25Xgap>[125X [27XOrder( Product( res.gens ) );[127X[104X
    [4X[28X5[128X[104X
    [4X[25Xgap>[125X [27Xres:= RunBBoxProgram( check.program, "dummy", res.gens, rec() );;[127X[104X
    [4X[25Xgap>[125X [27XIsBound( res.gens );  IsBound( res.result );[127X[104X
    [4X[28Xfalse[128X[104X
    [4X[28Xtrue[128X[104X

  [1X6.3-4 [33X[0;0YCriteria Used to Compute Minimality Information[133X[101X
  
  [33X[0;0YThe information about the minimal degree of a faithful [13Xmatrix representation[113X
  of  [22XG[122X  in  a  given  characteristic  or  over  a  given  field  in  positive
  characteristic  is derived from the relevant (ordinary or modular) character
  table  of  [22XG[122X, except in a few cases where this table itself is not known but
  enough information about the degrees is available in [HL89] and [Jan05].[133X
  
  [33X[0;0YThe  following  criteria  are  used  for  deriving  the  minimal degree of a
  faithful  [13Xpermutation  representation[113X  of  [22XG[122X from the information in the [5XGAP[105X
  libraries of character tables and of tables of marks.[133X
  
  [30X    [33X[0;6YIf the name of [22XG[122X has the form [10X"A[110X[22Xn[122X[10X"[110X or [10X"A[110X[22Xn[122X[10X.2"[110X (denoting alternating and
        symmetric  groups,  respectively) then the minimal degree is [22Xn[122X, except
        if [22Xn[122X is smaller than [22X3[122X or [22X2[122X, respectively.[133X
  
  [30X    [33X[0;6YIf  the  name  of  [22XG[122X has the form [10X"L2([110X[22Xq[122X[10X)"[110X (denoting projective special
        linear  groups  in  dimension  two)  then the minimal degree is [22Xq + 1[122X,
        except if [22Xq âˆˆ { 2, 3, 5, 7, 9, 11 }[122X, seeÂ [Hup67, Satz II.8.28].[133X
  
  [30X    [33X[0;6YIf  the  largest  maximal subgroup of [22XG[122X is core-free then the index of
        this  subgroup  is  the  minimal  degree.  (This  is used when the two
        character  tables  in  question  and the class fusion are available in
        [5XGAP[105X's  Character  Table  Library  ([Bre04]);  this  happens  for  many
        character tables of simple groups.)[133X
  
  [30X    [33X[0;6YIf  [22XG[122X  has a unique minimal normal subgroup then each minimal faithful
        permutation representation is transitive.[133X
  
        [33X[0;6YIn  this  case,  the  minimal degree can be computed directly from the
        information  in  the table of marks of [22XG[122X if this is available in [5XGAP[105X's
        Library of Tables of Marks ([NMP11]).[133X
  
        [33X[0;6YSuppose  that  the  largest maximal subgroup of [22XG[122X is not core-free but
        simple  and normal in [22XG[122X, and that the other maximal subgroups of [22XG[122X are
        core-free.  In  this  case,  we take the minimum of the indices of the
        core-free  maximal  subgroups  and of the product of index and minimal
        degree  of  the  normal  maximal  subgroup.  (This  suffices  since no
        core-free  subgroup of the whole group can contain a nontrivial normal
        subgroup of a normal maximal subgroup.)[133X
  
        [33X[0;6YLet  [22XN[122X be the unique minimal normal subgroup of [22XG[122X, and assume that [22XG/N[122X
        is  simple  and has minimal degree [22Xn[122X, say. If there is a subgroup [22XU[122X of
        index [22Xn â‹… |N|[122X in [22XG[122X that intersects [22XN[122X trivially then the minimal degree
        of [22XG[122X is [22Xn â‹… |N|[122X. (This is used for the case that [22XN[122X is central in [22XG[122X and
        [22XN Ã— U[122X occurs as a subgroup of [22XG[122X.)[133X
  
  [30X    [33X[0;6YIf  we  know a subgroup of [22XG[122X whose minimal degree is [22Xn[122X, say, and if we
        know either (a class fusion from) a core-free subgroup of index [22Xn[122X in [22XG[122X
        or  a  faithful permutation representation of degree [22Xn[122X for [22XG[122X then [22Xn[122X is
        the  minimal  degree  for  [22XG[122X. (This happens often for tables of almost
        simple groups.)[133X
